puppeteer-sample
===

## Description

sample project and sample web app for Puppeteer and CodeceptJS.

## Install 

```
$ cd ./puppeteer-sample
$ npm install
```

## Usage

* Start Web App.

```
$ cd ./sample-web-app
$ php -S localhost:8000
```

* Run CodeceptJS.

```
$ cd ./puppeteer-sample
$ ./node_modules/codeceptjs/bin/codecept.js run --steps
```
